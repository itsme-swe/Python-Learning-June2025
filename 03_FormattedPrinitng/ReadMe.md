### ASCII Code { American Standard For information Interchange }

1. **A - Z** ⇒ 65 90
2. **a - z** ⇒ 97 - 122
3. **A - Z** ⇒ 48 - 57

### Function to retrieve ASCII value of character

1.      ord('A')   # ASCII value of 'B' is 66

2.      chr(98)   # Char associated with ASCII value 101 is 'e'
